CHARACTER SWAP TRAINER RE4 UHD

The best modification ever created for Resident Evil 4 (2005)
On March 13 2023, I released my best work on RE4.

The Trainer

"New trainer introduction" was the title I gave the thread when I was creating it. Showing step by step feature by feature available in the trainer.

The main feature of this trainer is being able to play the complete game using all the characters available in this game without worrying about crashes or other types of problems.
There was a previous ways, but it was completely precarious and it was impossible to make this change without having problems during the campaign, for example:

Before my trainer:
Cutscenes didn't work;
When opening the inventory the character became invisible;
When equipping certain weapons the game would crash;
The game randomly closed several times during gameplay;
You couldn't save your progress, otherwise the game would crash.;
You can't manage Ashley (Wait and Follow-me);
And more, and more.

Impactful Features

Knife Parry

We've brought the newly implemented Knife Parry system introduced in the remake, to the classic.
This is the most used feature in the last days.

Walk through cutscenes

Breaking the game in a level where you can freely walk through the scenes
In this video I was also testing the mercenaries in the scenes, both new features for this update.

Breaking the game even more.

Melees changing in real time

Doing this was previously impossible due to constant crashes. No more!

Ada with cinematic physics

Original Ada has boring physics in her dress, with the trainer we added the same physics she has inside the scenes.
I don't need to mention this was impossible before and no one ever had this idea either.
